# 🎮 Nitro Games - Portal de Evolução dos Videogames

Este é um projeto de portal web moderno desenvolvido para explorar a história e a evolução tecnológica dos consoles das gigantes **Sony, Microsoft e Nintendo**. 

O projeto foi desenvolvido como parte do meu portfólio acadêmico e pessoal, unindo paixão por jogos e aprendizado em desenvolvimento front-end.

## 🚀 Tecnologias Utilizadas
* **HTML5**: Estruturação semântica de todas as páginas.
* **CSS3**: Layout moderno com Flexbox, Variáveis CSS para temas dinâmicos e efeitos Neon.
* **Font Awesome & Google Fonts**: Para tipografia profissional e ícones sociais.
* **Design Responsivo**: Adaptado para diferentes tamanhos de tela.

## 🎨 Diferenciais do Projeto
* **Temas Dinâmicos**: Cada marca (Xbox, PlayStation, Nintendo) possui sua própria identidade visual com cores personalizadas que mudam dinamicamente.
* **UX/UI Interativo**: Menu de navegação que reage com a cor característica de cada marca ao passar o mouse.
* **Conteúdo Curado**: Linha do tempo detalhada e textos justificados para uma leitura clara e profissional.
